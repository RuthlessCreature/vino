# UI language

## 总体风格

- 黑色哑光底
- 细线框
- 低饱和青蓝高亮
- 高信息密度，但层级清楚
- 绝不做花哨动画优先于可读性

## iPhone 端

- 预览全屏
- 控件半透明悬浮
- 参数编辑以纵向卡片组织
- 重点状态做胶囊标签：在线、录制、推理、保存、模型

## Desktop 端

- 左侧设备列表
- 中央设备详情 / 参数控制
- 下方数据终端
- 右侧批处理与模型管理

## 颜色建议

- 背景：`#050608`
- 主面板：`#0C1014`
- 描边：`#24303A`
- 强调：`#62F0FF`
- 警告：`#FFB347`
- 错误：`#FF5A6B`
- 成功：`#55E39E`
